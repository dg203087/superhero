# Superhero

Welcome to the Superhero Gem! This Ruby Gem provides a CLI to view stats and info about popular superheroes across several different comic universes. 

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'superhero'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install superhero

## Usage

After checking out the repo, run bin/setup to install dependencies. 

To install this gem onto your local machine, run bundle exec rake install. To release a new version, update the version number in version.rb, and then run bundle exec rake release, which will create a git tag for the version, push git commits and tags, and push the .gem file to rubygems.org.

## Development

After checking out the repo, run `bin/setup` to install dependencies. Then, run `rake spec` to run the tests. You can also run `.bin/superhero` for an interactive prompt that will allow you to experiment.

To install this gem onto your local machine, run `bundle exec rake install`. To release a new version, update the version number in `version.rb`, and then run `bundle exec rake release`, which will create a git tag for the version, push git commits and tags, and push the `.gem` file to [rubygems.org](https://rubygems.org).
